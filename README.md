## Recent Work

**[intelliTrip](https://intellitrip-iota.vercel.app)** — AI travel planner with a dual-provider LLM backend (Groq + Ollama), real-time SSE streaming, and a custom token-overlap deduplication pass to keep itineraries consistent. React/TypeScript + Node.js/Express.

**[Welcome to America](https://welcome-to-america.vercel.app)** — Immigration resource finder. Interviewed immigrants first, rewrote the information architecture based on what they said, then built it. Searchable directory of 50+ organizations, filterable by category and ZIP, with multilingual support.
